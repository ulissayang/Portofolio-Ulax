# Portfolio Ulis Leuwol (Update Terbaru)

Ini adalah source code terbaru website portofolio yang sudah ditambahkan pengalaman MagangHub di Lapas Saparua dan section Keterampilan.

## Cara Menggunakan (Untuk Deploy ke Cloudflare)

1. Ekstrak file zip ini di laptop Anda.
2. Buka folder hasil ekstrak di VS Code.
3. Buka Terminal di VS Code (`Ctrl + ~`).
4. Instal dependencies dengan mengetik:
   `npm install`
5. Lakukan Build untuk Cloudflare dengan mengetik:
   `npm run build`
6. Tunggu hingga selesai. Sebuah folder bernama **`dist`** akan muncul di sebelah kiri.
7. Buka Cloudflare Pages (seperti yang Anda tunjukkan di screenshot).
8. Tarik (Drag) folder **`dist`** tersebut dan lepaskan (Drop) ke kotak upload Cloudflare.
9. Website Anda akan otomatis terupdate ke versi yang paling baru ini!
